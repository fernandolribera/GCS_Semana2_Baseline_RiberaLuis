1. El sistema debe permitir a los usuarios registrarse con su numero de cedula y contraseña
2. El sistema debe permitir a los usuarios iniciar sesión con sus credenciales.
3. El sistema debe permitir a los usuarios recuperar su contraseña.
4. El sistema debe enviar un correo de confirmación para el registro.

# Requisitos no funcionales
1. El sistema debe permitir el acceso solo a usuarios autenticados.
2. La información de los usuarios debe almacenarse de forma segura.
